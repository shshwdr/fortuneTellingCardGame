PopupDialog Prefab Structure Guide
=====================================

To create the PopupDialog prefab in Unity:

1. Create an empty GameObject named "PopupDialog"
2. Add Canvas component (if not under a Canvas already)
3. Add the PopupDialog script component
4. Add a Canvas Group component for fade effects

Structure:
PopupDialog (PopupDialog script, Canvas Group)
├── Background (Image with semi-transparent dark color)
├── Panel (Image for the popup background)
│   ├── Title (TextMeshPro - UI) [Reference to titleText]
│   ├── Message (TextMeshPro - UI) [Reference to messageText]  
│   ├── ButtonContainer (Horizontal Layout Group)
│   │   ├── Button1 (Button) [Reference to button1]
│   │   │   └── Text (TextMeshPro - UI) [Reference to button1Text]
│   │   └── Button2 (Button) [Reference to button2]
│   │       └── Text (TextMeshPro - UI) [Reference to button2Text]

Set references in PopupDialog script:
- menu = Background
- animatedRect = Panel
- titleText = Title TextMeshPro component
- messageText = Message TextMeshPro component
- button1 = Button1 Button component
- button2 = Button2 Button component
- button1Text = Button1's Text TextMeshPro component  
- button2Text = Button2's Text TextMeshPro component

Save as a prefab in Resources/Prefabs/PopupDialog.prefab

ShopMenu Prefab Structure Guide (Updated)
==========================================

To create the ShopMenu prefab in Unity:

1. Create an empty GameObject named "ShopMenu"
2. Add the ShopMenu script component
3. Add a Canvas Group component

Structure:
ShopMenu (ShopMenu script, Canvas Group)
├── Background (Image with semi-transparent dark color)
├── Panel (Image for the shop background)
│   ├── Title (TextMeshPro - UI displaying "Shop")
│   ├── MoneyDisplay (TextMeshPro - UI) [Reference to moneyText]
│   ├── ScrollView (Scroll View)
│   │   └── Content
│   │       └── ItemContainer (Vertical Layout Group) [Reference to itemContainer]
│   └── NextDayButton (Button) [Reference to nextDayButton]
│       └── Text (TextMeshPro - UI displaying "Proceed to Next Day")

ShopItem Prefab Structure (NEW)
===============================

Create a unified ShopItem prefab:

ShopItem (ShopItem script)
├── DisplayContainer (Empty GameObject) [Where item-specific displays go]
├── PurchaseButton (Button) [Reference to purchaseButton in ShopItem script]
│   └── Text (TextMeshPro - UI) [Reference to purchaseButtonText in ShopItem script]

RuneDisplay Prefab (NEW)
========================

Create a specific rune display prefab:

RuneDisplay
├── NameText (TextMeshPro - UI)
├── DescriptionText (TextMeshPro - UI)
├── CostText (TextMeshPro - UI)
├── RuneIcon (Image) [Optional rune-specific icon]
└── EffectPreview (TextMeshPro - UI) [Optional effect description]

CardDisplay Prefab (NEW)
========================

Create a specific card display prefab:

CardDisplay
├── NameText (TextMeshPro - UI)
├── DescriptionText (TextMeshPro - UI)  
├── CostText (TextMeshPro - UI)
├── CardArtwork (Image) [Optional card artwork]
└── EffectPreview (TextMeshPro - UI) [Optional effect description]

ShopMenu Script References:
- itemContainer = ItemContainer transform
- shopItemPrefab = ShopItem prefab
- runeDisplayPrefab = RuneDisplay prefab
- cardDisplayPrefab = CardDisplay prefab
- moneyText = MoneyDisplay TextMeshPro component
- nextDayButton = NextDayButton Button component
